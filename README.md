# bfx-facs-db-sqlite

## Config

### name

Used to construct the database name: sqlite_$NAME_$LABEL.db

### label

Used to construct the database name: sqlite_$NAME_$LABEL.db


### Example

**db-sqlite.config.json**

```json
{
  "sqlite": {
    "name": "foo",
    "label": "bar"
  }
}

```
